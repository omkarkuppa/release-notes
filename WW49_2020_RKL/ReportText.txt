[CPU]

  [CBnT][ADL+][RKL][SACM] Implement new NEM size computation method
  
  BIOS MUST apply below new formula for the effective LLC size and NEM size calculation for non power of 2 core skus (6core and 10core sku). This formula can be common for power of 2 core skus.
  
  LLC size for NEM = (LLC size / Num of Slices) x (2 ^ [log2 (Num of Slices)])
  
  Based on CPUID EAX=4, ECX=3 call we get:
  a)  LLC Cache Size in Bytes = (Ways + 1) * (Partitions + 1) * (Line_Size + 1) * (Sets + 1) = (EBX[31:22] + 1) * (EBX[21:12] + 1) * (EBX[11:0] + 1) * (ECX + 1)
  b)  Way size = LLC size /  Ways;
  c)  #CBOs = MSR_UNC_CBO_CONFIG[5:0]        // Get #CBOs from MSR_UNC_CBO_CONFIG (0x396 for client, 0x2FFE for server)
  d)  CBO Way size = Way size / #CBOs
  e)  #Effective CBOs = floor (#CBOs, 2**N)  // Closest 2**N below #cores
  f)  Effective LLC size = LLC size * #Effective CBOs / #CBOs        // Proportionally reduced
  g)  NEM size = Effective LLC size - #Effective CBOs * CBO Way size = (LLC size - Way size ) * (#Effective CBOs / #CBOs)
