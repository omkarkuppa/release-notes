  [MTL] EC: Enlarge ECMD stall time
  
  [Issue Description]
  In the CMDR check, the check loop is 5000 and the delay between every
  check is 100 us. Which means at least timeout value is 0.5 second.

[SiliconPkg]

  WrRetrainOvrd = 0 Set On Last SAGV Point Only
  
  [Issue Description]
  A change was implemented in MRC 0.7.27.7 to keep WrRetrainOvrd=1 until
  the MrcNormalMode function is executed. This change was added to resolve
  a fast boot issue seen where initcomplete_wr is not asserted on the MC
  when Write Retraining is enabled and InitCompleteOverride flow is run
  during frequency switching. This change did not take into account the
  fact that WrRetrainOvrd must be set to it's final value for each SAGV
  Point before the MemSS PMA MRC_SAVE bit. With this change, the
  WrRetrainOvrd bit ends up changing from 0 to 1 after an SAGV Frequency
  switch to SAGV Points 0-2 effectively disabling Write Retraining.
  
  [Resolution]
  1. Reverted MRC 0.7.27.7 change
  2. WrRetrainOvrd is set to 1 before the InitCompleteOverride flow clears
  the InitCompleteOverride enable bits in the PHY. The original
  WrRetrainOvrd value is restored after the InitCompleteOverride enable
  bits are cleared.
  
  Package/Module: ClientOneSiliconPkg/IpBlock/MemoryInit/Mtl
